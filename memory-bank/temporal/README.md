# Temporal Notes

Temporary workspace for session notes, debugging.

## File Types
- `debug-[issue].md` - Debugging notes
- `scratch-[topic].md` - Quick development notes

## Usage
- Create temporary notes here
- Migrate important findings to permanent memory bank files
- Delete when no longer needed

## Lifecycle
1. Create temporal notes
2. Develop content
3. Migrate to permanent locations (features/, core files, etc.)
4. Archive/delete
